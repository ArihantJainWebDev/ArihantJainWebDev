## Selected Work

### Hub0.live
Anonymous real-time voice chat platform built with WebRTC, low-latency peer-to-peer calling, and O(1) matchmaking logic. It was designed for fast anonymous session pairing and got real users through campus outreach with zero marketing budget — which is a polite way of saying the product had to earn attention on its own.  

### Drapels.com
AI-powered developer education SaaS with career roadmaps, DSA practice tooling, AI interview simulation, and ATS resume features. I built and launched the product end-to-end, onboarded users, and learned firsthand that product quality alone does not magically solve distribution, no matter how emotionally attached a founder is to the dashboard.  

### AI Companion System
A personal research project built around a dual-LLM architecture using one model for real-time interaction and another for asynchronous background memory tasks. It includes RAG with ChromaDB, episodic memory, fact extraction, dream generation, browser control, and async memory operations to avoid slowing down chat.  

### EXUR
A gamified adaptive learning platform built during Smart Indore Hackathon 2026 in an 8-hour on-site sprint. It tracked cognitive performance in real time, adjusted difficulty dynamically, and generated topic-wise roadmaps with weak/moderate/strong classification plus external resource recommendations.  

### Proposal Generator
An AI-powered business tool that generates tailored client proposals using Llama 3.3 70B through Groq, with smart case-study matching, real-time preview, and PDF export. It’s one of those tools that exists because writing repetitive proposals is a deeply unromantic use of developer time.
